Welcome to FISH ISCN Nomenclature Generator (FISHING)

The interface is aiming to simplify the generation of an ISCN compliant annotation for break-apart FISH probe counts obtained from interphase hybridization. After selecting the test, the interface will ask for signal counts from atleast n=50 nuclei. Using the ISCN nomenclature, the interface will transform these counts and generate the probe specific annotation.
Note: The interface is considered a decision support tool to simplify FISH Break-Apart annotation.
